# مینی‌اپ تلگرام - راهنمای نصب سریع

## گام 1: نصب وابستگی‌ها

\`\`\`bash
cd miniapp-src
npm install
\`\`\`

## گام 2: اجرای محیط توسعه

\`\`\`bash
npm run dev
\`\`\`

سپس مرورگر را باز کنید: http://localhost:5173

## گام 3: ساخت نسخه تولید

\`\`\`bash
npm run build
\`\`\`

فایل‌های نهایی در `../app` قرار می‌گیرند.

## گام 4: تنظیم بات تلگرام

1. به [@BotFather](https://t.me/BotFather) در تلگرام مراجعه کنید
2. دستور `/newapp` را ارسال کنید
3. بات خود را انتخاب کنید
4. اطلاعات مینی‌اپ را وارد کنید:
   - عنوان: نام مینی‌اپ
   - توضیحات: توضیح کوتاه
   - عکس: لوگو (640x360)
   - GIF/ویدیو: پیش‌نمایش (اختیاری)
   - URL: `https://yourdomain.com/app/`

5. دکمه Web App را به بات اضافه کنید:
   - دستور `/setmenubutton` را ارسال کنید
   - URL مینی‌اپ را وارد کنید

## نکات مهم

- فایل‌های فونت را در `app/fonts/` قرار دهید
- مطمئن شوید PHP نسخه 7.4+ است
- HTTPS برای تلگرام الزامی است
- از BotToken برای احراز هویت استفاده کنید

## مشکلات رایج

### 1. npm install خطا می‌دهد
- نسخه Node.js را چک کنید: `node --version` (باید 16+ باشد)
- Cache را پاک کنید: `npm cache clean --force`

### 2. Build موفق نیست
- پوشه `node_modules` را حذف کنید
- دوباره `npm install` کنید

### 3. تلگرام مینی‌اپ را نشان نمی‌دهد
- SSL را چک کنید (باید HTTPS باشد)
- URL را در BotFather بررسی کنید
- Telegram script را در HTML چک کنید

### 4. API کار نمی‌کند
- مسیر proxy در `vite.config.js` را چک کنید
- فایل `api/miniapp.php` را بررسی کنید
- خطاهای PHP را در error_log چک کنید
